# Changelog

## 1.1.1 (2026-02-07)
- Fixed packers not properly inserting multiple stacks into a drawer
- Improved left click extraction behaviour
- Improved contraption handling (Minecart contraptions)

## 1.1.0 (2026-02-01)
- Updated Settings GUI buttons to be more inline with Create's style
- Add support for Mechanical Arms to directly interact with Drawers
  - At the moment, players can't directly insert a mechanical arm into a drawer. Might make a config option or something in the future for this.
- Add slowness debuff when carrying multiple drawers that has items stored inside them
  - Can be configured in the config (turn off, enable/disable for Creative mod, and how many you can have before slowness kicks in)

## 1.0.0-beta (2026-01-10)
Initial release
